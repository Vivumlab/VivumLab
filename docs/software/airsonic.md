## SETUP

### Enabling Airsonic

#### Command:

**`vlab set airsonic.enable True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
airsonic
  enable: True
```

#### Finalising changes:

run: **`vlab update_one service=airsonic`**
