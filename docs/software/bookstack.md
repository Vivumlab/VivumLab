## SETUP

### Enabling bookstack

#### Command:

**`vlab set bookstack.enable True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
bookstack
  enable: True
```

#### Finalising changes:

run: **`vlab update_one service=bookstack`**
