## SETUP

### Enabling calibre

#### Command:

**`vlab set calibre.enable True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
calibre
  enable: True
```

#### Finalising changes:

run: **`vlab update_one service=calibre`**
