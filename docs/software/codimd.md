## SETUP

### Enabling codimd

#### Command:

**`vlab set codimd.enable True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
>codimd
  enable: True

#### Finalising changes:

run: **`vlab update_one service=codimd`**

## OPTIONS

### HTTPS_ONLY
*Default: False*
*Options: True/False*

#### Command:

**`vlab set codimd.https_only True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
codimd
  https_only: True
```

##### Finalising changes:

run: **`vlab update_one service=codimd`**

### AUTH
*Default: False*
*Options: True/False*

#### Command:

**`vlab set codimd.auth True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
codimd
  auth: True
```

##### Finalising changes:

run: **`vlab update_one service=codimd`**

### DOMAIN
*Default: False*
*NOTE: include the sitename and top level domain suffix. eg. name.com, site.net*

#### Command:

**`vlab set codimd.domain codimd.com`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
codimd
  domain: codimd.com
```

##### Finalising changes:

run: **`vlab update_one service=codimd`**

### SUBDOMAIN
*Default: codimd*
*NOTE: Periods/ delimiters are not required. eg. 'media' will set the full URL as 'media.{{domain}}'*

#### Command:

**`vlab set codimd.subdomain media`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
codimd
  subdomain: media
```

##### Finalising changes:

run: **`vlab update_one service=codimd`**

### VERSION
*Default: {{  codimd.version  }}*
*NOTE: Ensure that the version exists*

#### Command:

**`vlab set codimd.version 2.7`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
codimd
  version: 2.7
```

##### Finalising changes:

run: **`vlab update_one service=codimd`**
