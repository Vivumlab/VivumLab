## SETUP

### Enabling digikam

#### Command:

**`vlab set digikam.enable True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
digikam
  enable: True
```

#### Finalising changes:

run: **`vlab update_one service=digikam`**
