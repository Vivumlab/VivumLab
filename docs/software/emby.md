## SETUP

### Enabling emby

#### Command:

**`vlab set emby.enable True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
emby
  enable: True
```

#### Finalising changes:

run: **`vlab update_one service=emby`**

## OPTIONS

**`vlab set emby.https_only True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```

  https_only: True
```

##### Finalising changes:

run: **`vlab update_one service=emby`**
