## SETUP

### Enabling ethercalc

#### Command:

**`vlab set ethercalc.enable True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
ethercalc
  enable: True
```

#### Finalising changes:

run: **`vlab update_one service=ethercalc`**

```

  https_only: True
```
