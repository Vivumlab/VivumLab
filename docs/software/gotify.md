## SETUP

### Enabling gotify

#### Command:

**`vlab set gotify.enable True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
gotify
  enable: True
```

#### Finalising changes:

run: **`vlab update_one service=gotify`**

## OPTIONS

### HTTPS_ONLY
*Default: False*
*Options: True/False*

#### Command:

**`vlab set gotify.https_only True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
gotify
  https_only: True
```

##### Finalising changes:

run: **`vlab update_one service=gotify`**

### AUTH
