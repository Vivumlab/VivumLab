## SETUP

### Enabling healthchecks

#### Command:

**`vlab set healthchecks.enable True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
healthchecks
  enable: True
```

#### Finalising changes:

run: **`vlab update_one service=healthchecks`**
