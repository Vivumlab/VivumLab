## SETUP

### Enabling homebridge

#### Command:

**`vlab set homebridge.enable True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
homebridge
  enable: True
```

#### Finalising changes:

run: **`vlab update_one service=homebridge`**

## OPTIONS

### HTTPS_ONLY
*Default: False*
*Options: True/False*

#### Command:

**`vlab set homebridge.https_only True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
homebridge
  https_only: True
```

##### Finalising changes:

run: **`vlab update_one service=homebridge`**

### AUTH
*Default: False*
*Options: True/False*

#### Command:

**`vlab set homebridge.auth True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
homebridge
  auth: True
```

##### Finalising changes:

run: **`vlab update_one service=homebridge`**

### DOMAIN
*Default: False*
*NOTE: include the sitename and top level domain suffix. eg. name.com, site.net*

#### Command:

**`vlab set homebridge.domain homebridge.com`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
homebridge
  domain: homebridge.com
```

##### Finalising changes:

run: **`vlab update_one service=homebridge`**

### SUBDOMAIN
*Default: homebridge*
*NOTE: Periods/ delimiters are not required. eg. 'media' will set the full URL as 'media.{{domain}}'*

#### Command:

**`vlab set homebridge.subdomain media`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
homebridge
  subdomain: media
```

##### Finalising changes:

run: **`vlab update_one service=homebridge`**
