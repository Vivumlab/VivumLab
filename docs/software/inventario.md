## SETUP

### Enabling inventario

#### Command:

**`vlab set inventario.enable True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
inventario
  enable: True
```

#### Finalising changes:

run: **`vlab update_one service=inventario`**
