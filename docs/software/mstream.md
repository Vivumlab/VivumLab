## SETUP

### Enabling mstream

#### Command:

**`vlab set mstream.enable True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
mstream
  enable: True
```

#### Finalising changes:

run: **`vlab update_one service=mstream`**
