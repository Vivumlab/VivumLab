## SETUP

### Enabling netdata

#### Command:

**`vlab set netdata.enable True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
netdata
  enable: True
```

#### Finalising changes:

run: **`vlab update_one service=netdata`**

### HTTPS_ONLY
*Default: False*
*Options: True/False*

#### Command:

**`vlab set netdata.https_only True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```

  https_only: True
```

##### Finalising changes:

run: **`vlab update_one service=netdata`**
