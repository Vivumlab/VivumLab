## SETUP

### Enabling openvpn

#### Command:

**`vlab set openvpn.enable True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
openvpn
  enable: True
```

#### Finalising changes:

run: **`vlab update_one service=openvpn`**

## OPTIONS

### HTTPS_ONLY
*Default: False*
*Options: True/False*

#### Command:

**`vlab set openvpn.https_only True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
openvpn
  https_only: True
```

##### Finalising changes:

run: **`vlab update_one service=openvpn`**

*Default: False*

*Default: openvpn*
