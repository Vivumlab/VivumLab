## SETUP

### Enabling ownphotos

#### Command:

**`vlab set ownphotos.enable True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
ownphotos
  enable: True
```

#### Finalising changes:

run: **`vlab update_one service=ownphotos`**

```

  https_only: True
```

*Default: False*

*Default: ownphotos*
