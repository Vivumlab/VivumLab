## SETUP

### Enabling pihole

#### Command:

**`vlab set pihole.enable True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
pihole
  enable: True
```

#### Finalising changes:

run: **`vlab update_one service=pihole`**
