## SETUP

### Enabling readarr

#### Command:

**`vlab set readarr.enable True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
readarr
  enable: True
```

#### Finalising changes:

run: **`vlab update_one service=readarr`**

  https_only: True
