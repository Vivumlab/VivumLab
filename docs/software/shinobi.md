## SETUP

### Enabling shinobi

#### Command:

**`vlab set shinobi.enable True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
shinobi
  enable: True
```

#### Finalising changes:

run: **`vlab update_one service=shinobi`**
