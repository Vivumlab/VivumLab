## SETUP

### Enabling sickchill

#### Command:

**`vlab set sickchill.enable True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
sickchill
  enable: True
```

#### Finalising changes:

run: **`vlab update_one service=sickchill`**
