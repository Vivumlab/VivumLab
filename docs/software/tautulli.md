## SETUP

### Enabling tautulli

#### Command:

**`vlab set tautulli.enable True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
tautulli
  enable: True
```

#### Finalising changes:

run: **`vlab update_one service=tautulli`**

### HTTPS_ONLY
*Default: False*
*Options: True/False*

#### Command:

**`vlab set tautulli.https_only True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```

```

run: **`vlab update_one service=tautulli`**
