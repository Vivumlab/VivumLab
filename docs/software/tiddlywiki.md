## SETUP

### Enabling tiddlywiki

#### Command:

**`vlab set tiddlywiki.enable True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
tiddlywiki
  enable: True
```

#### Finalising changes:

run: **`vlab update_one service=tiddlywiki`**
