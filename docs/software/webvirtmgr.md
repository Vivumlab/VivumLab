## SETUP

### Enabling webvirtmgr

#### Command:

**`vlab set webvirtmgr.enable True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
webvirtmgr
  enable: True
```

#### Finalising changes:

run: **`vlab update_one service=webvirtmgr`**
