## SETUP

### Enabling wekan

#### Command:

**`vlab set wekan.enable True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
wekan
  enable: True
```

#### Finalising changes:

run: **`vlab update_one service=wekan`**
