## SETUP

### Enabling barcodebuddy

#### Command:

**`vlab set barcodebuddy.enable True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
barcodebuddy
  enable: True
```

#### Finalising changes:

run: **`vlab update_one service=barcodebuddy`**
