## SETUP

### Enabling beets

#### Command:

**`vlab set beets.enable True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
beets
  enable: True
```

#### Finalising changes:

run: **`vlab update_one service=beets`**
