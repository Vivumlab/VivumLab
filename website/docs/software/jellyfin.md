## SETUP

### Enabling jellyfin

#### Command:

**`vlab set jellyfin.enable True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
jellyfin
  enable: True
```

#### Finalising changes:

run: **`vlab update_one service=jellyfin`**
