## SETUP

### Enabling piwigo

#### Command:

**`vlab set piwigo.enable True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
piwigo
  enable: True
```

#### Finalising changes:

run: **`vlab update_one service=piwigo`**
