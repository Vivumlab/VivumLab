## SETUP

### Enabling qbittorrent

#### Command:

**`vlab set qbittorrent.enable True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
qbittorrent
  enable: True
```

#### Finalising changes:

run: **`vlab update_one service=qbittorrent`**
