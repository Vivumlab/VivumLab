## SETUP

### Enabling ubooquity

#### Command:

**`vlab set ubooquity.enable True`**

#### File alteration:

set the appropriate service settings in `settings/config.yml` to true

eg.
```
ubooquity
  enable: True
```

#### Finalising changes:

run: **`vlab update_one service=ubooquity`**
